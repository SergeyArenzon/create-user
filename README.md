

## Installation

```javascript
  // 1: get to "/client" and run:
  npm i
  // 2: get to "/server" and run:
  npm i

```

## Usage

```javascript
// get to "/client"
// client running on localhost:5001
npm run dev

// get to "/server"
// server running on localhost:5001
npm start
```

You can can see the client on http://localhost:3000
